# 第十章 服务环境搭建

# 学习目标

熟练在服务器上安装 `JDK`

# 第一节 安装 `JDK`

> 第一步 查询系统中自带的 `JDK`

```shell
rpm -qa | grep jdk
```

> 第二步 卸载系统中自带的 `JDK`

```shell
rpm -e --nodeps jdk的rpm软件包的名字
```

> 第三步 将 `jdk.tar.gz` 上传到/`opt` 目录下

> 第四步 解压 `jdk` 压缩包

```shell
tar -zxvf jdk压缩包名称
```

> 第五步 在/`etc`/`profile.d` 目录下创建 `my`\`\_env.sh` 文件(系统不建议直接修改/`etc`/`profile`)

```shell
cd /etc/profile.d
touch my_env.sh
```

> 第六步 在 `my`\`\_env.sh`文件中配置`JAVA`\`\_HOME` 环境变量

```shell
#JAVA_HOME
JAVA_HOME=/opt/jdk-17.0.7
PATH=$PATH:$JAVA_HOME/bin
export PATH JAVA_HOME
```

> 第七步 执行 `source` /`etc`/`profile.d`/`my`\`\_env.sh`使环境变量立即生效，如果不生效就重启`java`  -`version`
